Open Photo AI

An open-source alternative to Topaz Photo AI.

Install

macOS & Linux

curl -fsSL https://vegidio.github.io/open-photo-ai/install.sh | sh

Windows (PowerShell)

irm https://vegidio.github.io/open-photo-ai/install.ps1 | iex

Pin a version

curl -fsSL https://vegidio.github.io/open-photo-ai/install.sh | OPAI_VERSION=26.3.0 sh
$env:OPAI_VERSION='26.3.0'; irm https://vegidio.github.io/open-photo-ai/install.ps1 | iex
